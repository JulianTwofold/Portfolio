Pokemon Daten von API nutzen

Nicht jedes Pokemon wird gleich geboren. Es gibt Gründe dafür, warum ein Glurak viel stärker ist als ein Karpador. Das liegt vor allem an den sogenannten Statuswerten, die ein Pokemon hat. Jedes Pokemon hat Werte für Angriff, Verteidigung und Geschwindigkeit. Je höher diese Werte, desto besser.

Wer ein starkes Team haben möchte, schaut deshalb zuerst auf die Pokemon mit den höchsten Statuswerten. Im Internet gibt es meiner Meinung nach nicht genug Funktionen, um diese zu sortieren. Deshalb habe ich diese Seite als Übung gemacht.

Meine Webseite erlaubt es, einige Werte auszuschalten. Das ist zum Beispiel nützlich, wenn man das stärkste langsame Pokemon herausfinden möchte. Der tiefere Angriffswert wird automatisch ausgeschalten, da es redundant ist und nicht der Gesamtstärke hilft.

Programmiert wurde die Webseite mit HTML, CSS, TypeScript und mithilfe der Pokemon API.

Das Git Repository zum Projekt befindet sich unter: https://github.com/JulianTwofold/RelevantPokemonStats

Wie ich das JSON file erstellt habe befindet sich unter: https://juliantwofold.github.io/createPokemonJson
